Table fact_ticket
Fact Table to hold data from tickets
* ticket_number text
  ticket_type integer
* customer_name text
* open_date integer
  close_date integer
* status text
* priority integer DEFO 9
  create_date date
  update_date date
Indexes
PkPk_ticket_data ON ticket_number
